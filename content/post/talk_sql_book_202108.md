  - 本の目次みて、やりたいこと近いことのページにあるSQL例を参考にちょっと書き換えてみるのがよさげ

- あと一発のSQLで書く必要はないので、小規模な集計を繰り返して組み合わせて行くほうがいい
  - よくWITH句で、小さい中間テーブルつくりまくって、最後にどかんってつなげてる

```
WITH
    hoge AS ( SELECT ...), 
    fuga AS ( SELECT ...)
SELECT .... FROM hoge, fuga ...
```
